## What’s New in Cryyer v0.6.1

We’ve been busy making Cryyer more reliable and flexible. This latest release focuses on making the tool easier to set up and more robust when handling your data.

### Key Improvements

- **Gist Subscriber Storage:** You can now manage your subscriber list using GitHub Gists. It’s a simple, secure way to handle your audience without needing a complex database setup.
- **Smarter AI Drafts:** We’ve improved how Cryyer talks to AI models. If a response isn't quite right the first time, the system now automatically retries, ensuring you get a usable draft every time.
- **Flexible Pipelines:** You now have more control over when updates are sent. Whether you want to trigger emails for every release, a weekly digest, or both, the new pipeline flags let you choose what works best for your project.
- **Faster Onboarding:** The initial setup is now much faster thanks to a non-interactive mode, making it easier to integrate Cryyer into your existing automation.
- **Refined Reliability:** We fixed several small bugs around permissions and badge rendering to make sure your project looks great on GitHub and your workflows run smoothly.

Thanks for using Cryyer to keep your community informed!
